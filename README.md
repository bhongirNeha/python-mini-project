# python-mini-project
#OBJECTIVE
Iris Dataset was explored using various tools.Overall, the visualizations and analysis provided information of the characteristics and variations of the Iris flower species.

#TOOLS USED
Python,numpy,pandas,matplotlib,plotly,pathlib

#Key Findings
The Key findings of this project was that the data was successfully analyzed using the above libraries.Visualizations using matplotlib and plotly was carried out.Graphs,interactive plots were generated.The graphs helped to analyze the distribution of the data.Several observations such as trend from line plot,correlation from scatter plot was noted.Simple mathematical logics such as mean,median,standard deviation of the data was found out to summarize the dataset.
